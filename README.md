# README

A quick orientation to the current files. More are being added over the coming days.

- **governanceREADME.md** — the record-level access governance design (model, grant table, lifecycle, enforcement, handoff requirements).
- **CDM_Governance_ERD** (`.png`) — entity-relationship diagram of the in-scope tables, custom extensions, and the governance table.
- **customOMOP-CDM.xlsx** — field-level data dictionary, one sheet per table plus a combined `Data_Dictionary` sheet. Descriptions are split into `User Guide` and `ETL Convention` columns. Canonical table descriptions are pulled directly from OHDSI -> https://ohdsi.github.io/CommonDataModel/cdm54.html#Current_Support_for_CDM_v54

Supporting material: `CDM_Governance.dbml` (source for the ERD)
